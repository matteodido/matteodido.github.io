Progetto n.1 con Arduino

Cestino automatico

Il primo progetto che sarà presente nel sito è anche il più recente, si tratta di un cestino con apertura del coperchio automatica, in base al passaggio o meno di un oggetto davanti al sensore.

i componenti utilizzati per questo progetto sono:

1) Arduino

2) Sensore ultrasuoni HC-SR04

3) Motore Servo

4) Breadboard o fili per il collegamento

Collegare i vari componenti come nello schema sottostante e caricare nella memoria del controllore il codice descritto sotto.

#include <Servo.h> //include la Libreria Servo.h

Servo myservo;//crea l’oggetto di tipo Servo, myservo sarà il motore utilizzato

int pos = 0;    //inizializza una variabile di tipo intero pos il cui valore sarà la posizione da impartire al servo
int trigPin = 9;    //Variabile che indica il pin di Arduino al quale è attaccato il pin Trigger del sensore ad ultrasuoni
int echoPin = 10;    //Variabile che indica il pin di Arduino al quale è attaccato il pin Echo del sensore ad ultrasuoni
long duration, cm, inches; //Variabili per misurare la distanza dell'oggetto rilevato 
int CestStatus = 0; //stato del cestino (0 chiuso / 1 aperto)
 
void setup() {
  Serial.begin (9600); //Inizia il collegamento alla porta seriale
  //Define inputs and outputs
  pinMode(trigPin, OUTPUT); //Definisce i pin di INPUT e OUTPUT
  pinMode(echoPin, INPUT);
  myservo.attach(8); // Lega l’oggetto myservo al pin a cui è stato collegato il servo
  myservo.write(pos); //Sposta il motore nella posizione iniziale pos=0
}
 
void loop() {
  // Il sensore è attivato da pulsazioni HIGH di 10 microsecondi o più
  // Da un piccolo segnale LOW prima, per assicurare un impulso HIGH pulito
  digitalWrite(trigPin, LOW);
  delayMicroseconds(5);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
 
  // Legge il segnale dal sensore e calcola il tempo trascorso dall'invio al ritorno
  pinMode(echoPin, INPUT);
  duration = pulseIn(echoPin, HIGH);
 
  // Converte il tempo in distanza
  cm = (duration/2) / 29.1;     // Si divide per 29.1 o moltiplica per 0.0343
  inches = (duration/2) / 74;   // Si divide per 74 o si moltiplica per 0.0135
  
  //Stampa nel monitor seriale la distanza dell'oggetto in cm
  Serial.print(cm); 
  Serial.print("cm");

  if (cm < 15){  //Se l'oggetto si trova a una distanza < 15cm allora cambia lo stato del cestino e alza il coperchio
    Serial.print("Aperto");
    CestStatus == 1;
    myservo.write(pos+100);
  }
  else{  //Altrimenti chiude o tiene chiuso il cestino
    delay(1000); //tempo in più per togliere la mano dal cestino
    Serial.print("Chiuso");
    CestStatus == 0;
    myservo.write(pos);
  }
  delay(200); //tempo tra un refresh delle misure e l'altro
}
