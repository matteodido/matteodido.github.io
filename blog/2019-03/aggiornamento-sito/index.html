Aggiornamento sito
  • 5 Mar, 2019
  • Lettura di un minuto

Guida per aggiornare il sito tramite comandi git

  1. Cancellare il contenuto della cartella "username.github.io"
  2. Aprire un terminale e nella cartella "username.github.io" eseguire il comando:
    >git pull origin master
    Il quale dovrebbe confermare che è tutto aggiornato
  3. Entrare nella cartella "nomesito" ed eseguire il comando:
    >hugo -d ../username.github.io/
    Per creare nella cartella "username.github.io" i file md necessari per il sito.
  4. Entrare ora nella cartella "username.github.io" e eseguire il comando:
    > git status
    Nel terminale apparirà un riassunto dei file da modificare nella repository
  5. eseguire il comando:
    > git add -all
    Per aggiornare i file contenuti nella repository.
  6. Eseguire il comando:
    > git commit -m "Commento sull'update"
    Per dare un commento all'update fatto al sito
  7. Eseguire il comando:
    > git push origin master
    Per caricare i file nella repository. Sarà richiesto l'username e la password dell'account github collegato.
