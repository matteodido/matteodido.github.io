Installazione di i3wm su una nuova macchina linux

Programmi necessari per l’installazione (consigliato installarli prima di i3wm):

  • Feh
  • Blueman
  • xbacklight
  • gtk-theme
  • lxappearance

Installazione e setup di i3wm:

  • Installare i3wm da terminale con il seguente comando:

sudo apt-get i3wm

Chiudere la sessione e rientrare selezionando i3wm come Desktop Enviorment.

  • Creare i file i3status.conf, config e lock.sh all’interno della cartella /home/“username”/.config/i3/ e copiare al loro interno il testo dei file sotto:

config i3status.conf lock.sh

  • Scaricare un immagine di sfondo (in questo link lo sfondo dello screenshot), rinominarlo “t.jpg” e posizionarlo nella cartella Scaricati del sistema.

  • Scaricare il tema del sistema da questo link e copiarlo nella cartella /usr/share/themes.

  • Scaricare i3-gapps e seguire le indicazioni della pagina GitHub per l’installazione e la configurazione.
